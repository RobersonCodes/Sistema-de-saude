# Inteligência Artificial

O sistema utiliza IA para interpretar mensagens do cidadão.

Objetivo:

identificar intenção
identificar especialidade médica
identificar data desejada
auxiliar agendamento automático

---

Exemplo

Entrada:

quero marcar consulta com cardiologista amanhã

Saída estruturada:

intencao: AGENDAR_CONSULTA
especialidade: CARDIOLOGIA
data: 2026-04-20
